# 리뷰 분석 가이드라인

## 분석 목표
- 고객 리뷰의 정확한 감정 분석
- 핵심 키워드 및 주제 추출
- 스팸/가짜 리뷰 탐지
- 셀러에게 유용한 인사이트 제공

## 감정 분석 기준

### 긍정적 리뷰 특징
- 만족, 추천, 좋음 등의 표현
- 재구매 의사 표현
- 타인 추천 의사
- 구체적인 장점 언급

### 부정적 리뷰 특징
- 불만, 실망, 나쁨 등의 표현
- 환불/교환 요구
- 타인에게 비추천
- 구체적인 문제점 지적

### 중립적 리뷰 특징
- 객관적 사실 나열
- 장단점 균형적 언급
- 감정 표현 최소화

## 키워드 추출 우선순위
1. 제품 품질 관련 키워드
2. 배송/서비스 관련 키워드
3. 가격/가성비 관련 키워드
4. 디자인/외관 관련 키워드
5. 사용성/기능 관련 키워드

## 스팸 탐지 기준
- 과도하게 짧거나 긴 리뷰
- 반복적인 문자/단어 사용
- 광고성 링크나 코드 포함
- 부자연스러운 표현 패턴
- 과도한 이모지 사용

## 주제 분류 카테고리
- **품질**: 제품 자체의 품질, 내구성, 성능
- **배송**: 배송 속도, 포장 상태, 배송 서비스
- **가격**: 가격 대비 만족도, 할인, 가성비
- **디자인**: 외관, 색상, 스타일, 미적 요소
- **사용성**: 사용 편의성, 기능성, 설치/조립
- **서비스**: 고객 서비스, AS, 문의 응답

## 분석 결과 활용 방안
- 제품 개선점 도출
- 마케팅 포인트 발견
- 고객 만족도 트렌드 파악
- 경쟁사 대비 강약점 분석
