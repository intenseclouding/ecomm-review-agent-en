# 셀러별 프롬프트 관리 가이드라인

## 프롬프트 구성 요소

### 기본 정보

- **셀러 ID**: 고유 식별자
- **브랜드명**: 브랜드 이름
- **업종**: 판매 제품 카테고리
- **타겟 고객층**: 주요 고객 연령대 및 특성

### 브랜드 톤앤매너

- **브랜드 성격**: 친근함, 전문성, 럭셔리 등
- **커뮤니케이션 스타일**: 격식, 캐주얼, 유머러스 등
- **핵심 가치**: 브랜드가 추구하는 가치
- **금지 표현**: 사용하지 말아야 할 단어나 표현

### 응답 템플릿 설정

#### 긍정적 리뷰 템플릿

```json
"positive_response_templates": [
    "기본 감사 표현",
    "만족 공감 표현",
    "지속 서비스 약속",
    "브랜드 특화 메시지"
]
```

#### 부정적 리뷰 템플릿

```json
"negative_response_templates": [
    "진심어린 사과",
    "개선 의지 표현",
    "해결 방안 제시",
    "재방문 유도 메시지"
]
```

#### 중립적 리뷰 템플릿

```json
"neutral_response_templates": [
    "의견 감사 표현",
    "참고 약속",
    "지속 개선 의지"
]
```

### 주제별 맞춤 설정

- **품질 관련 응답**: 품질 관리 철학, 개선 노력
- **배송 관련 응답**: 배송 정책, 포장 방식
- **가격 관련 응답**: 가격 정책, 가치 제안
- **디자인 관련 응답**: 디자인 철학, 개발 과정
- **서비스 관련 응답**: 서비스 정책, 고객 지원

### 개인화 설정

- **개인화 수준**: low, medium, high
- **키워드 활용**: 리뷰 키워드 반영 정도
- **감정 반영**: 고객 감정에 대한 반응 정도
- **맞춤 멘트**: 특별한 상황별 맞춤 메시지

## 셀러 프롬프트 예시

### 패션 브랜드 예시

```json
{
  "seller_id": "FASHION-001",
  "brand_name": "스타일리시",
  "category": "패션",
  "brand_personality": "트렌디하고 친근한",
  "communication_style": "캐주얼하면서도 세련된",
  "core_values": ["스타일", "품질", "개성"],
  "positive_response_templates": [
    "스타일리시한 선택 감사합니다! 만족스러운 스타일링 되셨길 바라요 ✨",
    "좋은 리뷰 정말 감사드려요! 앞으로도 트렌디한 아이템으로 찾아뵐게요 👗"
  ],
  "negative_response_templates": [
    "아쉬운 경험을 하셨군요 😔 더 나은 스타일과 품질로 보답하겠습니다",
    "소중한 피드백 감사해요. 스타일과 품질 모두 만족하실 수 있도록 개선하겠습니다"
  ]
}
```

### 전자제품 브랜드 예시

```json
{
  "seller_id": "TECH-001",
  "brand_name": "테크솔루션",
  "category": "전자제품",
  "brand_personality": "전문적이고 신뢰할 수 있는",
  "communication_style": "정중하고 전문적인",
  "core_values": ["혁신", "품질", "신뢰"],
  "positive_response_templates": [
    "제품에 만족해주셔서 감사합니다. 지속적인 기술 혁신으로 더 나은 제품을 제공하겠습니다.",
    "좋은 평가 감사드립니다. 앞으로도 최고 품질의 기술 제품으로 보답하겠습니다."
  ],
  "negative_response_templates": [
    "불편을 끼쳐드려 죄송합니다. 기술적 문제를 신속히 해결하고 개선하겠습니다.",
    "소중한 의견 감사합니다. 제품 품질과 기술력 향상에 더욱 노력하겠습니다."
  ]
}
```

## 프롬프트 관리 원칙

1. **일관성 유지**: 브랜드 아이덴티티와 일치
2. **정기 업데이트**: 시장 변화와 고객 피드백 반영
3. **A/B 테스트**: 다양한 템플릿의 효과 검증
4. **성과 모니터링**: 응답 효과성 지속 추적
5. **고객 반응 분석**: 고객 재반응 및 만족도 측정

## 프롬프트 최적화 방법

- **데이터 기반 개선**: 리뷰 분석 결과 활용
- **고객 피드백 반영**: 댓글에 대한 고객 반응 분석
- **경쟁사 벤치마킹**: 업계 모범 사례 참고
- **계절성 반영**: 시즌별 특화 메시지 적용
- **이벤트 연동**: 프로모션이나 이벤트와 연계
